Preguntas JavaScript

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

  • Semejanzas:

    Ambos son lenguajes de programación.

    Ambos utilizan sintaxis similar (llaves, estructuras de control, etc.).

  • Diferencias:

    Java es compilado y fuertemente tipado; JavaScript es interpretado y débilmente tipado.

    Java se usa principalmente para aplicaciones backend y móviles.

    JavaScript se ejecuta principalmente en el navegador.

¿Qué métodos tiene el objeto Date? (al menos 5)

  • getFullYear()
  • getMonth()
  • getDate()
  • getHours()
  • getTime()
  • setFullYear()

¿Qué métodos tienen los arreglos? (al menos 5)

  • push()
  • pop()
  • shift()
  • unshift()
  • slice()
  • splice()
  • forEach()
  • map()

¿Cómo se declara una variable con alcance local dentro de una función?

  • Usando let o const dentro de la función.

    Ejemplo: function ejemplo(){ let x = 10; }

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

  • Pueden ser modificadas desde cualquier parte del código.

    Generan errores difíciles de detectar.

    Rompen el principio de encapsulamiento.

    Hacen el mantenimiento más complicado.
