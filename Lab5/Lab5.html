Lab5: Frameworks de estilo

¿Qué es Bootstrap, cuáles son sus funciones y para qué sirve?

  • ¿Qué es Bootstrap?

    Bootstrap es un framework de desarrollo front-end de código abierto que facilita la creación de sitios web modernos y responsivos.

    Fue desarrollado originalmente por Twitter y permite diseñar interfaces web de manera rápida utilizando HTML, CSS y JavaScript.

  • Funciones principales:

    Incluye un sistema de rejilla (grid system) para organizar contenido en filas y columnas.

    Proporciona componentes prediseñados como botones, formularios, tarjetas, tablas, alertas y barras de navegación.

    Incluye clases utilitarias para aplicar estilos como colores, márgenes, alineación y espaciado.

    Cuenta con componentes interactivos que utilizan JavaScript, como modales, carruseles y menús desplegables.

  • ¿Para qué sirve?

    Sirve para desarrollar páginas web adaptables a diferentes dispositivos (responsive design).

    Permite ahorrar tiempo al usar estilos y componentes ya diseñados.

    Facilita mantener un diseño visual consistente en todo el sitio web.
